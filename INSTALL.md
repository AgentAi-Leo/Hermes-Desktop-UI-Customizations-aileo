# Install

1. Back up your Hermes Desktop source.
2. From `apps/desktop`, dry-run first:

```bash
bash /path/to/Hermes-Desktop-UI-Customizations/releases/2026-07-11/scripts/apply-customizations.sh --dry-run
```

3. Apply only if dry-run succeeds:

```bash
bash /path/to/Hermes-Desktop-UI-Customizations/releases/2026-07-11/scripts/apply-customizations.sh --apply
```
